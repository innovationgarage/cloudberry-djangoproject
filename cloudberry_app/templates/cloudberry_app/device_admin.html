    Where to buy devices

    To use this service you need to buy a physical device and install one of our images on it. You can select any of the devices and stores below:

    • OrangePI R1 (Two ethernet port version)

    How to write the image

    In the next step you'll get to download a device image. Once downloaded, you'll have to write this image to an SD card and then put the card in your device. The following is a short guide to how to do this:

    On Microsoft Windows

    Download Etcher and use it to write the image file.

    On Linux

    Find out which device file (/dev/sdX) corresponds to your sd-card reader by removing and inserting the card to see which device file appears/dissapears. Then run the commands (replace X in /dev/sdX with the letter you found out):

            dd if=downloadedimage.img of=/dev/sdX
        sync
